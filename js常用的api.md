## Math常用的方法
- `Math.abs()` 返回绝对值
- `Math.ceil(),Math.floor() `向上取整和向下取整
- `Math.max(),Math.min()` 最大值和最小值
- `Math.round()` 四舍五入
- `Math.random()` 随机数
- `Math.pow()` 指数运算
- `Math.sqrt()` 平方根
- `Math.log()` 自然对数
- `Math.exp()` e的指数


## Array常用的方法
- 数组的创建
```javascript
// 字面量
var arr = [1,2,3]; // [1,2,3]
// 构造方式
var arr = Array(); // []
var arr = Array(1,2,3); // [1,2,3]
var arr = Array(3); //[,,]
```
